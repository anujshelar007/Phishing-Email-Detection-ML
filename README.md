# Phishing Email Detection Using Machine Learning

## Overview

This project is a Machine Learning-based phishing email detection system developed using Python and Scikit-learn.

The system analyzes email text and cybersecurity-related features such as URLs, IP-based links, and suspicious phishing keywords to classify emails as:

- Phishing Email
- Safe Email

---

# Features

- Machine Learning classification using Logistic Regression
- TF-IDF text feature extraction
- URL detection
- IP-address URL detection
- Suspicious keyword analysis
- Confusion matrix visualization
- Accuracy evaluation
- Real-time custom email testing

---

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Regular Expressions (Regex)

---

# Project Workflow

1. Load email dataset
2. Clean dataset
3. Extract cybersecurity features
4. Convert email text into numerical vectors
5. Train machine learning model
6. Predict phishing or safe emails
7. Display accuracy and confusion matrix

---

# Cybersecurity Features

The project detects:

- Suspicious URLs
- IP-based phishing links
- Dangerous phishing keywords
- Multiple links in emails

---

# Model Performance

- Accuracy: 96.43%
